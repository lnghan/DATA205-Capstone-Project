# DATA205 Capstone Project - MC311 Analysis

## **OVERVIEW**

This project explores the MC311 dataset from data.montgomery. It specifically focuses on three service areas: Highway Services, Treasury, and Code Enforcement, as well as five randomly chosen ZIP code in Montgomery County, Maryland: 20833 (Brookeville), 20866 (Burtonsville), 20904 (Colesville/Silver Spring), 20906 (Eastern Silver Spring), 20853 (Rockville)

I aim to identify patterns in request demand, response time, area, and potential disparities. Ultimately, this project could help highlight opportunities for improving efficiency and equity in local government services.
