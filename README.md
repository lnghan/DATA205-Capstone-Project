## DATA205-Capstone-Project

#### This project explores the MC311 dataset from data.montgomery. It specifically focuses on three service areas: Highway Services, Tree Maintenance, and Code Enforcement. By examining response times and whether requests meet expected deadlines, I identified patterns in efficiency and complexity across these services. This matters because it shows how well local government is meeting residents’ needs. This project could help identify areas for improvement in service delivery and resource allocation.
